# **App Name**: Ethical Frontier

## Core Features:

- Resource Management: Track and manage country's resources including Economy, Education, Well-being, Popular Support, Hunger, and Military & Religion.
- Character Roles: Implement character roles with unique abilities, such as Minister of Education, Economy Manager, Agriculture, Religious, Influencer and Military Commander.
- Decision Cards: Generate dilemma cards that present players with ethical choices, each affecting resources and progress.
- Boss Encounters: Define boss encounters as obstacles that require specific resource levels to overcome.
- Progress Tracking: Display progress on a 20-step board, requiring ethical decisions and overcoming challenges to advance.
- Win Condition AI: Determine win conditions (collective and individual) based on resource levels and character objectives, deciding which conditions are met, if any. Acts as a tool for decision making regarding the distribution of Capital
- Turn management: Manage turns for multiple players, allowing each player to make a choice on the current dilemma.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to represent governance and stability.
- Background color: Light Blue Gray (#ECEFF1) to create a clean and non-intrusive backdrop.
- Accent color: Vibrant Orange (#FF9800) to highlight important decisions and interactions.
- Body and headline font: 'PT Sans' for clear readability and a modern touch.
- Use simple, clear icons to represent resources and character roles.
- Design a clear dashboard layout with easily readable resource bars and decision card displays.
- Incorporate subtle animations for resource updates and card reveals.
